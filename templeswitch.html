Temple Switch

Temple Switch is a 2D- and 3D-hybrid puzzle platformer game made as part of a group project. Set in a mysterious temple, the player has the power to switch between 2D and 3D perspectives, allowing them to progress through puzzles and platforming challenges.

The team included Jenna Hood, Tyler Evans, Lukas Brothwell and Yuxin Zheng. My role as Technical Lead involved overseeing the technical aspects, writing code and using the game engine to implement gameplay mechanics.

Download Report GitHub

Why?

This game was created for the Interactive Media Group Project module at the University of York.

Tools

The game was created using Unity game engine and the C# programming language, along with tools like GitHub and Miro for collaboration.

Outcome

This group project overall earned a 2:1 class grade (69%), while the digital experience itself received a 72% mark.
