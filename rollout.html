Rollout!

Rollout! is a physics-based ball-rolling platformer where the player will collect coins and power-ups to progress through a platforming challenge. The goal of the game is to reach the end in the fewest number of deaths possible, while also collecting coins along the way.

Download Report GitHub

Why?

This game was created for the Media Production for Interactive Environments module at the University of York.

Tools

The game was created using Unity game engine and the C# programming language.

Outcome

This project overall earned a First class grade (80%).
