Zombie Attack

Zombie Attack is a web-based top-down zombie shooter. The player is equipped with a gun and must escape the zombie-infested area. Players will find a number of collectibles as they progress, including keys and medkits. Keys will allow the player to unlock doors and medkits will restore health points. The player can also interact with and rescue a number of NPCs.

Report GitHub

Desktop only

This game is best experienced on a desktop or laptop screen. Please switch to a larger display to play Zombie Attack.

Why?

This game was created for the Programming for Digital Media (Part II) module at the University of York.

Tools

The game was created using Javascript and the p5.js library.

Outcome

This project overall earned a First class grade (86%).
