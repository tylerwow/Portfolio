Dead End

Dead End is a first-person zombie survival game where the player must venture through an abandoned city to repair their broken-down truck. The game has been developed as an outcome on the research of diegetic interfaces in interactive media. The game explores how traditional interfaces, such as the heads-up display (HUD) and other user interfaces can be displayed diegetically, investigating the impact on traditional game traditional game design choices and exploring how the player experience, primarily relating to usability, is impacted.

Download Report GitHub

Why?

This game was created for the Interactive Media Individual Project module at the University of York.

Tools

The game was created using the Unity game engine and C# programming language.

Outcome

This project earned a 1st class grade (76%), successfully executing the design and implementation of diegetic interfaces and game mechanics. Dead End was presented at the 2026 Creative Technologies Showcase.
